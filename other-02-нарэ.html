О собеседнице

Нарэ — зоопсихолог, работает с проблемным поведением собак в Ереване. Ведёт работу как напрямую с собаками, так и параллельно с их владельцами — без участия хозяев результата нет. Специализация: агрессия, страхи, социализационные проблемы. Имела опыт в кинологии, работает с различными специалистами (ветеринарами, кинологами, грумерами). Клиентов находит преимущественно через знакомых, а также через платформу Зорели (армянская платформа для специалистов с питомцами).

Что обсуждалось

1. Профессиональный контекст

  • Работала в кинологии, затем перешла в зоопсихологию
  • Работает с разными проблемами: агрессия, страхи, травмы (в т.ч. после войны — собака, пережившая обстрел, 3 года сидит в комнате)
  • Параллельная работа с владельцем обязательна — если хозяин не продолжает дома, работа «уходит в воду»
  • Типичный срок работы: от нескольких недель до 7+ месяцев при глубоких страхах
  • Публикуется и находит клиентов через Зорели (армянская платформа для специалистов по животным)

2. Работа с ветеринарами

  • У собак в её практике встречались эпилепсия и психиатрические проблемы (поведение как симптом)
  • Нашла хорошего ветеринара (Хофзэп) — после долгих поисков, не через соцсети, а через знакомых
  • Проблема: в Ереване только 1–3 клиники на слуху, выбирать сложно
  • Когда не знаешь город — очень трудно выбрать куда обращаться
  • Однажды пропустила прививку — не было цифрового напоминания

3. Работа с кинологами

  • Работает на стыке зоопсихологии и кинологии: сначала оценивает, потом передаёт кинологу
  • Важно понимать профиль кинолога: кто специализируется на агрессии, кто на маленьких проблемах — это разные специалисты
  • Онлайн-консультация с кинологом — не работает: кинолог должен видеть собаку вживую
  • Рекомендация: база кинологов с описанием профиля (что умеет, с чем работает)

4. Грумеры — рейтинги критичны

  • Слышала случаи гибели собак у грумеров — качество непредсказуемо
  • Рейтинги и отзывы на грумеров: «если будет рейтингом — будет проще»
  • Сейчас нет инструмента для проверки грумера до визита

5. Ситтеры и выгул

  • Работала со случаями, когда ситтер оставался с собакой (daycare-формат)
  • Профессиональные daycare-места часто имеют камеры — обязательное требование
  • В Ереване найти ситтера трудно; pet-friendly кафе/рестораны стали лучше за последние 5–6 лет
  • Хотела бы базу ситтеров с отзывами и личной историей — чтобы понять, «похож ли человек на меня»

6. Pet-friendly места

  • Раньше (5–6 лет назад) pet-friendly мест почти не было, сейчас лучше
  • Центральные места (Ground Siren, Martha и др.) уже pet-friendly
  • Хотела бы карту/базу pet-friendly мест — особенно актуально для незнакомых районов
  • Дополнительная идея: какие жилые дома и кварталы разрешают держать питомцев

7. GPS, чипы, идентификация

  • 3–4 клиента были недовольны чипированием (но большинство чипируют)
  • Вместо чипа некоторые используют «кулончики» с QR-кодом
  • Местные жители (армяне) часто не доверяют сканированию чипа
  • GPS не упоминался как часто используемый инструмент в её практике

8. Онлайн-консультации

  • Онлайн для ветеринара — да: хорошо подходит для первичных вопросов и симптомов
  • Онлайн для кинолога — нет: нужно видеть собаку вживую, онлайн-формат искажает процесс
  • Сама использовала онлайн-консультации редко — предпочитает личный контакт

9. Запись к специалисту

  • Ситуация: позвонила в клинику — сказали «сейчас есть время», приехала — оказалось занято
  • Хотела бы видеть доступные слоты визуально, без звонков
  • Онлайн-запись к ветеринару с выбором врача, дня и времени — «идеально»

10. Питание и контент

  • Гуглит специфические данные о питании (например, сколько воды нужно французскому бульдогу)
  • Хотела бы базу знаний о питании с иллюстрациями — не сухой текст
  • Есть породы с критическими особенностями: фрэнчи должны пить много воды, иначе проблемы
  • Диетологи для животных — считает полезным направлением

11. Магазин / агрегатор

  • Если бы агрегатор показывал, в каком магазине есть конкретный товар — использовала бы
  • Проблема: в Армении нет многих специфических товаров, приходится искать по одному
  • Идея: агрегировать магазины + заведения (кафе, ветклиники) в одном месте

12. Уведомления

  • Получает много нотификаций из разных приложений — перестала их читать
  • Нужен баланс: не слишком много уведомлений, иначе пользователь «выключается»

13. Донорство крови для собак

  • Недавний кейс: искала донора крови для собаки друга — оказалось очень сложно найти
  • Считает, что база доноров (собаки с определённой группой крови) была бы ценна
  • Сейчас это работает только через личные связи

1 🔴 Рейтинги специалистов — острая боль для профессионала Нарэ как зоопсихолог сама выбирает ветеринаров, кинологов, грумеров — видит проблему изнутри. Грумеры без рейтингов = риск гибели животного
2 🔴 Онлайн-запись к ветеринару с визуальным слотом — «идеально» Реальный негативный опыт: звонила, сказали «есть время», приехала — занято. Визуальный выбор слота решает проблему
3 🔴 База доноров крови — незакрытая боль Сама искала донора для собаки друга — очень сложно. Сейчас работает только через знакомых
4 🟡 История болезни и напоминания — реальный кейс пропущенной прививки «Если бы была нота — я бы не забыла». Цифровая история особенно важна при работе с несколькими специалистами
5 🟡 База ситтеров с личными историями — доверие важнее рейтинга «Хочу понять, похож ли этот человек на меня». Сухая оценка не даёт уверенности
6 🟡 Онлайн-консультация с ветеринаром — да; с кинологом — нет Принципиальное различие: ветеринар может работать по симптомам, кинолог обязан видеть собаку. Важно в UX не смешивать эти два формата
7 🟡 Карта pet-friendly мест и домов — запрос озвучен сама Не только кафе — но и жилые здания и кварталы, где разрешены питомцы. Особенно ценно для переезжающих
8 🟢 Уведомлений не должно быть много «Когда много — перестаю читать». Важный UX-принцип: нотификации должны быть точечными
9 🟢 Профиль кинолога важен: с чем именно работает Кинолог по агрессии ≠ кинолог по маленьким проблемам. База без специализации бесполезна
